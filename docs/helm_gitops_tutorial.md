# ⛵ 终极云原生兵器：Helm (微服务导弹发射车) 与 GitOps 安装指北

在之前的章节中，为了部署 `leaf-service-user` 和 `leaf-service-order`，我们需要分别去写几十行的 `deploy-xxx.yaml` 和 `svc-xxx.yaml`。假设微服务规模增加到 50 个角色（含网关、权限中心等），维护 100 多个独立 YAML 配置文件将是一场灾难！

为了解决这个问题，业界诞生了 **Helm**，并在此基础上演化出了终极运维模式：**GitOps（ArgoCD）**。

---

## 📦 1. 什么是 Helm？（微时代的包管理器）

如果说 Kubernetes 是操作系统，那么 **Helm 就是相当于 Windows 的 `exe 安装包` 或者 Linux 的 `apt-get` / `yum`**。

### 痛点：K8s 原生 YAML 的缺陷
原生的 K8s YAML 是极其僵硬的“死代码”。如果我们在测试环境和生产环境想部署相同的 `User服务`，但只是 MySQL 的 IP、JVM 内存、或者是副本数量 (Replicas) 不一样，我们居然要把几百行的 YAML 代码原封不动复制两份，稍有改动极易漏改或者错改！

### 魔法：Helm 的模板引擎
Helm 引入了伟大的 **“骨架 (Template) + 灵魂参数 (Values)”** 设计！
通过刚才我们的改造，您可以打开 `infra-deploy/helm/leaf-springcloud` 这个全新的目录。你看到了什么？
1. **`templates/deployment.yaml`**：这是一段被“抽干”的骨架代码。里面只写了 `{{ .image }}`，`{{ .containerPort }}` 这种空参变量，没有任何具体的业务信息。
2. **`values.yaml`**：这是所有灵魂注入的地方。在这个文件里，像配表一样齐刷刷排布了所有上百个微服务的 `名称、镜像名、参数环境`。

**结果是什么？**
我们只需要写**一遍最干练简洁的模板骨架**代码，然后通过一个几百行的 `values.yaml` 清单，执行：
```bash
helm install my-leaf-system infra-deploy/helm/leaf-springcloud
```
只需敲响这一个命令，就能在 1 秒内一键将 50 个微服务导弹全部准确地射入 K8s 集群！
这就是 Helm！这就是被称为架构师必备的“**一键部署发射车**”！

---

## 🤖 2. 什么是 GitOps？（自动化流水线的究极大招）

即使有了 Helm，我们还要每次自己在服务器上敲 `helm install` 吗？不！在超级企业里，由于权限隔离，开发和运维甚至连 K8s 环境或服务器的密码都没有。那代码写完了怎么上线？

**ArgoCD (GitOps 的终极推行者)** 强势登场。

### 过去时代（CI/CD 流水线，如 Jenkins）：
老王把测试好的代码合入 `master`，然后登录 Jenkins 网页，大喊一声：“执行 K8s 部署！”。Jenkins 帮您执行了一段连去目标服务器的 Shell 脚本，把 K8s 的镜像更新了。
**缺点：** Jenkins 必须被授予极其危险的直接操作服务器集群的 Root 权限！并且一旦目标服务器被别人私下手敲改动过，Jenkins 是死死的，它根本不知道系统已经被篡改过了。

### GitOps 时代（基于状态循环）：
核心思想极为霸气：**GitHub / Git 仓库是真理的唯一来源！**
安装了 `ArgoCD` 等软件在集群里，这是一种主动监控式的“幽灵特工”。
1. 您再也不需要去点 Jenkins，您只需把刚才的 `values.yaml` 修改一版如 `replicas: 10` 提交入库。
2. ArgoCD 一天 24 小时每隔 3 分钟自动去 GitHub 读取你的代码。
3. 它一旦眼尖的发现 GitHub 上写的应该是 10 个副本，而现实运行的集群只有 1 个副本。它连警告都不发出，**它会以完全静默、自主的方式，使用超级管理员权限去执行底层的抹平动作，硬生生拉起 9 个新的节点，强行让“现实世界”和您的“代码文件”一比一对齐！**

这就是目前阿里、Google 都在采用的最为严苛、最为安全的无人化云原生大阵：**GitOps 自动流水线**。由于一切基于 Git，如果您想退回昨天没有 Bug 的时光，只需在 Git 敲下一个 Reset。在 Git 退回的那一瞬间，整个服务器的业务线将如时光倒流般被 ArgoCD 给复原！

我们在本地已经为您准备好了基础的 Helm 安装包，为了不让臃肿的工具打乱你的体验，若您准备好感受这种弹药发射器，随时可以在装好的微环境里试验 `helm` 指令的威力！
